B) How to Edit HTML Files?

We have included 27 custom HTML templates like Home Page, About, Faq's, Team and Contact etc. Please open any HTML files with a text editor like Dreamweaver, Notepad or Notepad++ and edit any lines what you want.

C) How to Edit CSS Files?

We have included some custom CSS styles like style.css (default). Please open any CSS files with a text editor like Dreamweaver, Notepad or Notepad++ and edit any lines what you want. For example if you want to edit your banner color open style.css and look at ".banner-con" for banner image and change your image.

 

D) How to use Color Schemes?

Please open style.css file from Contrive/assets/css folder with a text editor and build your own colors. #ffffff this is our primary color, you can search and replace all to your new color code.

 

E) How to use On Load Animation?

We are aos.js on load animation for our website. you can edit them by simply adding or changeing the predefined classes name.
